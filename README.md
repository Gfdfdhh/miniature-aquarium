# miniature-aquarium
## Как выглядит проект:
![Мини-аквариум сделанный на pygame.](https://raw.githubusercontent.com/Gfdfdhh/miniature-aquarium/refs/heads/main/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%20(445).png)
## Идеи в проекте:
* Разные размеры и скины рыб.
* Рыбы двигаются со случайной скоростью.
* Небольшой шанс что рыбы пукнут.
* Есть микроскопический шанс на спавн морской или звездыили ракушки.
* На фоне плавают пузыри.
